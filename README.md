# Pathmark v0.7.26 — Navigation shell and Pathmark Home

Adds a signed-in Pathmark Home so users choose Planning, Nutrition or Finance from one operational hub. Workspaces now open in a focused shell with a simple return path to Pathmark Home, reducing the previous double-navigation effect between top-level sections and module sections. Theme, About & Privacy, Public Home/Download and Developer are moved into utility actions from the signed-in hub.

The public signed-out Home, Privacy Policy, Terms of Service and Google access pages remain available for OAuth and public review.
